БИЗНЕС-МОДЕЛЬ

Это принципиально новая бизнес-модель для инфраструктурных проектов, новый формат экономических отношений между участниками, построенный на принципах открытости и непредвзятости. Блокчейн используется в качестве системы дистрибуции технологии владельцем интеллектуальной собственности.

БАЗОВЫЙ ТОКЕН СЕТИ

Для учета операций, совершаемых внутри блокчейна, используется базовый токен сети, который применяется для выражения стоимости любых других цифровых активов, привязанных к различным ценностям (стоимость перевозки, оплата труда и т.д.)

ОПИСАНИЕ ОДНОГО УЗЛА ДЕЦЕНТРАЛИЗОВАННОЙ СИСТЕМЫ

Несколько слов об одном узле децентрализованной системы. И еще несколько слов.

Узел сети (нода) - сервер сети, обеспечивающий её функционирование

Каждый узел работает в автономном режиме и содержит полную информацию обо всех событиях в сети, непрерывно синхронизируясь с остальными узлами сети.

Отсутствие единого органа управления сетью (центра), который бы устанавливал правила и осуществлял централизованное управление. Взаимодействие объектов организовано на уровне программного кода системы.

  • Децентрализованное хранение и обработка информации.
  • Децентрализованное распределение и учет средств.
  • Децентрализованное управление (выбор пользователями представителей, обеспечивающих работоспособность сети)

АВТОМАТИЗАЦИЯ

Объекты физической инфраструктуры снабжены автоматикой, непрерывно фиксирующей их состояния, отражаемые в блокчейне в виде событий (транзакций) без участия пользователей. Каждый узел сети может работать автономно, а все транзакции между объектами осуществляются напрямую, без участия посредников.

УЧЕТ В БЛОКЧЕЙНЕ

Каждый объект и субъект в блокчейн-сети имеет собственную учетную запись (цифровой паспорт), отражающую количество токенов (баланс) и сведения об их движении в системе (список транзакций). Эмиссия и обращение токенов санкционируется блокчейном.

ДЕЦЕНТРАЛИЗОВАННОЕ
РАСПРЕДЕЛЕНИЕ

Распределение средств происходит децентрализованно: средства, списанные с баланса пользователя в момент получения им услуги, направляются в счет затрат согласно бизнес-плану: плановая амортизация, техобслуживание, дивиденды инвесторам и проч.
